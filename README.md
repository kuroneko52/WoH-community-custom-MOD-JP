# 恐怖の世界、World of Horror (WoH) の日本語ローカライズ
WoHの世界観に合わせつつ、文脈、ニュアンスを考慮したローカライズ

## ・オリジナルMODファイル
[modding.zip](https://wohgame.com/modding.zip)

<br>

## ・How to Install

1. リポジトリをクローン

2. `WoH-community-custom-MOD-JP/` のルートにある `.ito` ファイルを全て、customフォルダにコピーして上書き

<br>

## ・フォルダ説明

`modding/` <br>
オリジナルのMODファイルが入ってます

`original-ito/` <br>
オリジナルの `.ito` ファイルが入ってます

`instructions/` <br>
.ito ファイルの設定マニュアルが入ってます。<br>
これを読めば誰でも新しいイベントを追加、既存のイベントの編集が出来ます。

<br>

このリポジトリには上記URLからダウンロードしたmodding.zipファイルが含まれます。<br>
つまり、URLからわざわざ落とさなくてもこのリポジトリをクローンするか、ファイルを単一で落とすだけで入手できます。

このゲームのMOD開発についての詳しい情報は下記リポジトリを参照してください。

[WoH Community Modding Guide](https://github.com/Myonmu/WoH-Community-Modding-Guide)
