# 恐怖の世界、World of Horror (WoH) の日本語ローカライズ
WoHの世界観に合わせつつ、文脈、ニュアンスを考慮したローカライズ

## ・オリジナルMODファイル
[modding.zip](https://wohgame.com/modding.zip)

<br>

## ・How to Install

1. オリジナルMODファイルをダウンロードして設置

2. リポジトリをクローン

3. `WoH-community-custom-MOD-JP/` のルートにある `.ito` ファイルを全て、customフォルダにコピーして上書き

4. `pareidolia` フォルダを作成して `awake_in_bed.png` をその中へ移動

<br>

## ・オリジナルMODファイル補足説明

`instructions.txt`

.ito ファイルの設定マニュアル<br>
これを読めば誰でも新しいイベントを追加、既存のイベントの編集が出来ます。

このリポジトリには含まれていません。

<br>

このゲームのMOD開発についての詳しい情報は下記リポジトリを参照してください。

[WoH Community Modding Guide](https://github.com/Myonmu/WoH-Community-Modding-Guide)
